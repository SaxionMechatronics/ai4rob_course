# AI for Robotics Course

A course covering AI for robotics, which includes modules specifically for foundations, deep learning, and reinforcement learning.

## Getting Started

### Prerequisites

- [Pixi](https://pixi.sh) package manager
- macOS, Linux, or Windows

### Installation

1. Clone the repository:
```bash
git clone git@bitbucket.org:mechatronica/ai4rob_course.git
cd ai4rob_course
```

2. Install dependencies using pixi:
```bash
pixi install
```

### Development

To start the local development server:

```bash
pixi run serve
```

This will start the MkDocs development server at `http://127.0.0.1:8000/`

### Build

To build the static site:

```bash
pixi run build
```

The built site will be in the `site/` directory.
